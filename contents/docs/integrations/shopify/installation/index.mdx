Installing the Weevio Shopify Widget is a straightforward process that typically takes 15-30 minutes. This guide will walk you through the three main steps: preparing your prerequisites, configuring the widget in Weevio Cloud, and adding the code to your Shopify theme.

## Installation Overview

The installation process consists of three main steps:

1. **Prerequisites Setup** - Create a Shopify Custom App and gather API credentials
2. **Weevio Cloud Configuration** - Connect your Shopify store and enable features
3. **Theme Integration** - Add the widget code to your Shopify product pages

## Before You Begin

Make sure you have:

- Admin access to your Shopify store
- Access to your Weevio Cloud account
- Ability to edit your Shopify theme code
- About 15-30 minutes to complete the setup

## Step 1: Prerequisites

Before installing the widget, you'll need to set up a Shopify Custom App with the appropriate API access.

**What you'll need:**
- Shopify Custom App with Admin API access
- Admin API access token
- Storefront API access token (optional, for improved variant selection)

[Learn how to set up prerequisites →](/docs/integrations/shopify/installation/prerequisites)

[Screenshot: Shopify Custom App creation screen]

## Step 2: Configure in Weevio Cloud

Once you have your Shopify API credentials, connect your store in Weevio Cloud and configure your desired features.

**In this step you'll:**
- Enter your PIMS settings
- Connect your Shopify store
- Configure order sync webhooks
- Purchase and enable widget features
- Copy your personalized widget code

[Learn how to configure Weevio Cloud →](/docs/integrations/shopify/installation/configuration)

[Screenshot: Weevio Cloud Shopify integration page showing store connection]

## Step 3: Add Widget to Theme

With your widget code copied from Weevio Cloud, you'll paste it into your Shopify theme's product template.

**In this step you'll:**
- Access your Shopify theme code editor
- Locate your product template file
- Paste the widget code
- Save and publish your changes
- Test the widget on a product page

[Learn how to add widget to theme →](/docs/integrations/shopify/installation/configuration#theme-integration)

[Screenshot: Shopify theme editor showing product template with widget code]

## Verification

After installation, verify the widget is working correctly:

1. Visit a product page in your live store
2. You should see shipping estimates or availability information
3. Open browser console (F12) and check for errors
4. Try changing variants to see the widget update
5. (If enabled) Test cross-sell by adding a product to cart

[Screenshot: Successful widget installation showing all features working]

## Common Issues

If the widget doesn't appear or isn't working correctly:

- Verify your Team ID and Shopify domain are correct in the widget code
- Check that API credentials are valid (test connection in Weevio Cloud)
- Ensure the widget container `<div id="Weevio-eCommerce-Widget-Root">` exists
- Check browser console for JavaScript errors
- Verify features are purchased in Weevio Cloud

For detailed troubleshooting, see the [Troubleshooting Guide](/docs/integrations/shopify/troubleshooting).
