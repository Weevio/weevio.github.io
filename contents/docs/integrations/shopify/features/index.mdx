The Weevio Shopify Widget offers seven powerful features that can be enabled individually or combined to create a premium shopping experience. Each feature is designed to solve specific e-commerce challenges and can be configured to match your business needs.

## Available Features

### 1. Improved Variant Selection

Transform your product variant selector with an enhanced, card-based interface that shows price differences and prevents invalid combinations.

**Benefits:**
- Better user experience than default Shopify selectors
- Shows price differences between variants at a glance
- Prevents customers from selecting invalid combinations
- Keyboard navigation for accessibility

[Learn more about Improved Variant Selection →](/docs/integrations/shopify/features/variant-selection)

[Screenshot: Card-based variant selector showing color and size options]

---

### 2. Shipping Estimates & Availability

Display accurate delivery estimates and real-time stock availability to help customers make informed purchase decisions.

**Benefits:**
- Reduces cart abandonment with transparent shipping info
- Shows local pickup options and same-day availability
- Displays real-time inventory across multiple sources
- Available in Basic (static) or Premium (zip-based) plans

[Learn more about Shipping Estimates →](/docs/integrations/shopify/features/shipping-estimates)

[Screenshot: Shipping estimates showing delivery timeframes and local pickup]

---

### 3. In-Stock Recommendations

Automatically suggest alternative products when items are out of stock, keeping customers engaged and preventing lost sales.

**Benefits:**
- Prevents lost sales from out-of-stock items
- Helps customers find similar products instantly
- Shows price comparisons and availability
- Full-screen modal with up to 5 alternatives

[Learn more about In-Stock Recommendations →](/docs/integrations/shopify/features/recommendations)

[Screenshot: Modal showing in-stock alternative products]

---

### 4. Cross-Sell Recommendations

Display complementary products after customers add items to their cart, increasing average order value.

**Benefits:**
- Increases average order value
- Shows relevant product associations
- One-click add-to-cart for cross-sell items
- Supports B2B customer filtering

[Learn more about Cross-Sell →](/docs/integrations/shopify/features/cross-sell)

[Screenshot: Cross-sell modal with complementary products]

---

### 5. B2B Wholesale Pricing

Display negotiated wholesale pricing for logged-in business customers automatically, with "Your Price" badges and discounts.

**Benefits:**
- Supports B2B and B2C customers on one platform
- Shows customer-specific pricing automatically
- Displays discount amounts and savings
- Integrates with Shopify B2B features

[Learn more about B2B Pricing →](/docs/integrations/shopify/features/b2b-pricing)

[Screenshot: Product page showing B2B pricing with discount badge]

---

### 6. Multi-Source Inventory

Aggregate stock from multiple sources (Shopify, Ingram Micro, TD SYNNEX) to show accurate availability and reduce out-of-stock situations.

**Benefits:**
- Accurate inventory across multiple vendors
- Reduces out-of-stock scenarios
- Configurable vendor thresholds
- Automatic inventory aggregation

[Learn more about Multi-Source Inventory →](/docs/integrations/shopify/features/multi-source-inventory)

[Screenshot: Inventory display showing stock from multiple sources]

---

### 7. PunchOut Commerce

Seamless integration with SAP Ariba, Coupa, and other procurement systems, allowing business buyers to shop without logging in.

**Benefits:**
- Streamlines B2B procurement workflows
- No login required for PunchOut users
- Supports major procurement platforms
- Session-based authentication with 12-hour expiration

[Learn more about PunchOut Commerce →](/docs/integrations/shopify/features/punchout)

[Screenshot: PunchOut session showing customer-specific pricing]

## Feature Combinations

Features can be enabled individually or combined for maximum impact:

### B2C Store (Recommended)
- ✅ Improved Variant Selection
- ✅ Premium Shipping Estimates
- ✅ In-Stock Recommendations
- ✅ Cross-Sell

### B2B Wholesale Store
- ✅ Improved Variant Selection
- ✅ Basic Shipping Estimates
- ✅ B2B Wholesale Pricing
- ✅ Cross-Sell
- ✅ Multi-Source Inventory

### Hybrid B2B/B2C Store
- ✅ All Features Enabled
- Automatically adapts based on customer type

### Enterprise with PunchOut
- ✅ All Features
- ✅ PunchOut Commerce
- Optimized for procurement workflows

## Feature Requirements

Different features have different requirements:

| Feature | Requires Purchase | Additional Requirements |
|---------|------------------|------------------------|
| Improved Variant Selection | ❌ | Storefront API token |
| Basic Shipping ETAs | ✅ | None |
| Premium Shipping ETAs | ✅ | USPS API credentials |
| In-Stock Recommendations | ✅ | Shipping ETAs enabled |
| Cross-Sell | ✅ | PIMS product associations |
| B2B Pricing | ✅ | Shopify B2B API key, price lists |
| Multi-Source Inventory | ✅ | Vendor API credentials |
| PunchOut Commerce | ✅ | Customer tag configuration |

## Enabling Features

Features are controlled by:

1. **Feature Purchases** - Purchase features in Weevio Cloud settings
2. **Feature Flags** - Enable in widget configuration
3. **API Credentials** - Configure required integrations

See the [Configuration Guide](/docs/integrations/shopify/configuration) for detailed setup instructions.

## Performance Impact

The widget is designed for optimal performance:

- **Lazy loading** - Features load only when needed
- **Request debouncing** - Prevents excessive API calls
- **Caching** - Session data cached for faster subsequent loads
- **Abort controllers** - Cancels stale requests automatically

## Customization Options

Each feature can be customized:

- **Visual styling** - Inherits your theme styles
- **Messaging** - Customize text and labels
- **Behavior** - Configure thresholds and rules
- **Integration** - Connect with your existing systems

See [Advanced Configuration](/docs/integrations/shopify/configuration/advanced) for customization options.
