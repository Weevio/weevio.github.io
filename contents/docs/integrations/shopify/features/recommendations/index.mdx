The In-Stock Recommendations feature automatically suggests alternative products when the selected item is out of stock or has low inventory, helping customers find what they need and preventing lost sales.

[Screenshot: In-stock recommendations modal showing alternative products]

## What It Does

When a product is out of stock or has limited inventory, the widget displays a "Similar in-stock options available" link. Clicking it opens a full-screen modal with up to 5 alternative products that:

- Are currently in stock
- Match the original product type/category
- Show price comparisons
- Display shipping estimates
- Offer one-click selection

## Benefits

### For Your Customers

- **Find alternatives instantly** - No need to search your entire catalog
- **Stay informed** - Know which similar products are available
- **Compare easily** - See price differences and features side-by-side
- **Keep shopping** - Don't leave your site when first choice unavailable
- **Save time** - Curated suggestions vs. manual searching

### For Your Business

- **Prevent lost sales** - Turn out-of-stock situations into purchases
- **Reduce bounce rate** - Keep customers engaged when products unavailable
- **Increase customer satisfaction** - Help customers find solutions
- **Lower support burden** - Fewer "when will this be back?" inquiries
- **Improve inventory flow** - Move stock from similar products

## How It Works

[Screenshot: Flow diagram showing out-of-stock → recommendations link → modal → selection]

1. **Detection** - Widget identifies product is out of stock or low inventory
2. **Link appears** - "Similar in-stock options available" link displays
3. **Customer clicks** - Full-screen modal opens with alternatives
4. **Browse options** - Customer reviews up to 5 similar products
5. **Selection** - Click on alternative to view that product page
6. **Continue shopping** - Customer proceeds with in-stock alternative

## Requirements

To use In-Stock Recommendations:

- **Feature purchase:** `integration_shopify_in_stock_variant_recommendation`
- **Shipping estimates enabled** - Basic or Premium plan
- **Backend configuration** - Recommendations sourced from product type/category

## Configuration

### Enabling the Feature

The feature is automatically enabled when:

1. Feature is purchased in Weevio Cloud
2. Shipping Estimates feature is active
3. Backend has recommendation logic configured

No additional configuration required in widget code.

### How Recommendations Are Selected

The backend selects alternatives based on:

- **Product type match** - Same product type in Shopify
- **Category similarity** - Similar product categories
- **Stock availability** - Must be in stock
- **Price range** - Similar price points preferred
- **Customer tags** - B2B filtering if applicable

## Triggering Conditions

Recommendations appear when:

- **Out of stock** - Product or variant unavailable
- **Low inventory** - Below configured threshold
- **Discontinued** - Product marked as discontinued
- **Preorder** - Long lead time before availability

Conditions configured in Weevio Cloud settings.

## Performance

The feature is optimized for speed:

- **Lazy loading** - Modal loads only when clicked
- **Image optimization** - Compressed product images
- **API caching** - Brief cache of recommendation data
- **Async rendering** - Doesn't block page load

**Typical performance:**
- Link display: `<50ms`
- Modal open: 200-400ms
- Alternative loading: 300-500ms

## Troubleshooting

### No Recommendations Showing

**Check:**
- Feature is purchased in Weevio Cloud
- Shipping estimates are enabled
- Product has product type set in Shopify
- Similar products exist and are in stock

### Wrong Alternatives Displayed

**Check:**
- Product types match correctly
- Categories configured properly
- Inventory is accurate
- Recommendation rules in Weevio Cloud

### Modal Not Opening

**Check:**
- No JavaScript errors in console
- Link element is clickable (no CSS conflicts)
- Modal container renders correctly
- Browser JavaScript is enabled

### Slow Loading

**Check:**
- Alternative product images optimized
- API response times acceptable
- Network conditions
- Browser performance

See the [Troubleshooting Guide](/docs/integrations/shopify/troubleshooting) for more solutions.
